Project Overview

The objective of this project is to automatically identify the genre of a music track using machine learning and deep learning techniques. The pipeline combines feature extraction, dataset augmentation, synthetic mashups, and convolutional neural networks for robust classification performance.

Audio Feature Extraction

Extracts meaningful spectral representations from raw audio signals using mel spectrograms and preprocessing techniques suitable for CNN-based learning.

Data Augmentation

Improves model generalization by generating noisy audio variations and synthetic mashups to create a larger and more diverse training dataset.

Deep Learning Architecture

Uses a Convolutional Neural Network designed for audio spectrogram classification with efficient training and inference pipelines.

Inference Pipeline

Performs predictions on unseen music files and generates genre distributions using a trained deep learning model.

Complete Workflow

The notebook implements a complete deep learning lifecycle starting from raw audio processing to deployment-ready inference.

1. Raw Audio Collection

Music files are collected and organized genre-wise for preprocessing and training.

2. Feature Extraction

Audio waveforms are transformed into mel spectrograms and structured tensors that can be consumed by neural networks.

3. Synthetic Data Creation

Noise injection and mashup generation techniques are used to create a more robust dataset capable of handling real-world variations.

4. CNN Training

A convolutional neural network is trained on the processed spectrogram data using PyTorch.

5. Validation & Evaluation

Model performance is validated using train-validation splits and genre distribution analysis.

6. Inference

The trained model predicts genres for unseen songs and outputs final predictions efficiently on GPU hardware.
